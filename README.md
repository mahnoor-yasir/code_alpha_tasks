# CodeAlpha Frontend Development Tasks

## Overview

This repository contains the frontend development tasks completed as part of the CodeAlpha Frontend Development Internship. Each project is built using modern web technologies with a focus on responsive design, clean user interfaces, and interactive functionality.

## Projects

### 1. Image Gallery
A responsive image gallery with an elegant user interface.

**Features**
- Responsive grid layout
- Image categories and filters
- Lightbox image preview
- Previous and next navigation
- Hover animations
- Smooth transitions
- Mobile-friendly design

**Technologies**
- HTML5
- CSS3
- JavaScript

---

### 2. Professional Calculator
A fully functional calculator designed for students, professionals, and everyday users.

**Features**
- Basic arithmetic operations
- Scientific calculations
- Percentage calculations
- Parentheses support
- Memory functions
- Square, cube, roots
- Trigonometric functions
- Logarithmic functions
- Theme switching
- Keyboard shortcuts
- Responsive interface
- Calculation history

**Technologies**
- HTML5
- CSS3
- JavaScript

## Getting Started

1. Clone the repository.

```bash

```

2. Open the desired project folder.

3. Launch the `index.html` file in your web browser.

No additional dependencies or installation are required.
